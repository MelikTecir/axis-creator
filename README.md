# axis-programcreator
Haftalık ders çalışma programınızı saniyeler içinde hazırlamanızı, özelleştirmenizi ve profesyonel bir çıktı (PDF) almanızı sağlayan kullanıcı dostu bir web uygulamasıdır. Özellikle Polinom konseptine uygun tasarımıyla, öğrencilerin haftalık hedeflerini net bir şekilde görmesini sağlar.
